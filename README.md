## Introduction:
> The Elderly Care offers information's about the elderly health and notify them about there medicines and appointments in shape of notifications.
## Features:
- user can Sign in, with Google.
- user can display Medicine Page, Appointment Page.
- user can add, edit , delete and display From Medicine and Appointments pages.
- User can display the health States for elderly  and set Notifications for medicine or appointment.
## Technologies Used:
- Firebase(Fire Store, Firebase Authentication)
- Data Binding.
- Navigation.
- Broadcast Receiver → Notification.
- Activities.
- Fragment. 
- Layout.
- Animations → Lottie.
